ReactAuth

Simple user authentication
for React apps

ReactAuth is a JavaScript library for adding user authentication to React apps. Add logic for login, signup, social login, sessions, etc. in a few minutes. 

          $ npm install react-auth
Star Fork

What is ReactAuth?

ReactAuth is a module that provides routes and components for React that will allow you to solve common user management tasks such as login and signup.

Install

You can install ReactAuth using npm:

$ npm install react-auth --save

Usage

Include ReactAuth and its components into your app:

// Using ES6.
import ReactAuth, { Router, AuthenticatedRoute, LoginLink } from 'react-auth';

// Not using ES6.
var ReactAuth = require('react-auth').default;
var Router = require('react-auth').Router;
var AuthenticatedRoute = require('react-auth').AuthenticatedRoute;
var LoginLink = require('react-auth').LoginLink;

// Or if you are using it from a browser without UMD.
var Router = ReactAuth.Router;
var AuthenticatedRoute = ReactAuth.AuthenticatedRoute;
var LoginLink = ReactAuth.LoginLink;
