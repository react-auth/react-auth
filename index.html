What is ReactAuth?

ReactAuth is an extendable, simple to use authentication module for React for login (username/password and social), registration, forgotten password, and more.

Install

The easiest way to get ReactAuth is by running one of the following commands:

// Bower
bower install react-auth

// npm
npm install react-auth --save

If installed via Bower, include the following script tag:

<script src="bower_components/react-auth/react-auth.min.js"></script>

Usage

1. Include ReactAuth and its components into your app:

// Using ES6.
import ReactAuth, { LoginForm } from 'react-auth';

// Not using ES6.
var ReactAuth = require('react-auth').default;
var LoginForm = require('react-auth').LoginForm;

// Or if you are using it from a browser without UMD.
var LoginForm = ReactAuth.LoginForm;

2. Initialize the module:

ReactAuth.init({
  // See the Configuration section for how to configure ReactAuth.
});

3. Add components to your page:

class LoginPage extends React.Component {
  render() {
    return (
      <div id="login-page">
        <LoginForm />
      </div>
    );
  }
}

Configuration

Below is a complete list of all configuration options.

// Configure ReactAuth with your own AuthProvider.
ReactAuth.init({
  authProvider: customAuthProvider
});

// Configure ReactAuth with a specific/supported AuthProvider.
// Supported providers currently are 'stormpath' or 'default'.
ReactAuth.init({
  authProvider: {
    use: 'stormpath'
  }
});

Components

  • Authenticated — Show content only when logged in.
  • NotAuthenticated — Show content only when logged out.
  • SocialLoginLink — Link to login with a social provider, such as Facebook.
  • SocialLoginButton — Button to login with a social provider, such as Facebook.
  • LogoutLink — Link to end a session and log out.
  • LoginForm — Component to create a login form.
  • SignupForm — Component to create a signup form.
  • ResetPasswordForm — Component to create a reset password form.
  • ChangePasswordForm — Component to create a change password form.
  • UserProfileForm — Component to create a user profile form.
  • VerifyEmailView — Component to create a view for email verification.
